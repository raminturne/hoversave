HoverSave

Save images with a hotkey

Active
Press S while hovering an image
Tip: Alt+Shift+S toggles on/off from anywhere.

Click the box, then press a key. Works on any keyboard layout (English, Farsi, Arabic, etc.).

We'll create (or reuse) a HoverSave subfolder inside whatever you pick — no need to make one yourself.

No folder picked — saving to Chrome's default Downloads.

Hover any image on any page and press your key. Files keep their original name and extension.

EN: Windows won't let Chrome select Desktop (or Documents/Downloads/Pictures/Music/Videos) directly. In the dialog, open that folder, click New folder, name it (e.g. HoverSave), then select the new subfolder.

فارسی: ویندوز اجازه نمی‌ده Chrome مستقیماً پوشه‌ی Desktop (یا Documents/Downloads/Pictures/Music/Videos) رو انتخاب کنه. توی همون دیالوگ، اول اون پوشه رو باز کن، دکمه‌ی New folder رو بزن، یه اسم بذار (مثلاً HoverSave)، و همون زیرپوشه‌ی جدید رو انتخاب کن.
